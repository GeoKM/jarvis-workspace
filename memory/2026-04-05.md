# 2026-04-05

## Incident

- During Proxmox skill setup, `~/.config/openclaw/openclaw.env` was edited unsafely with a targeted replacement.
- The edit dropped the `OPENROUTER_API_KEY` entry.
- Result: `openclaw-gateway` failed to start correctly and entered a restart loop.
- Keith manually repaired `openclaw.env`; gateway service recovered.

## Lesson

- Treat `~/.config/openclaw/openclaw.env` as a critical secrets file.
- Always back it up before any modification.
- Always read the full file before editing.
- Prefer append-only or full-file verified rewrites over fragile targeted replacements.
- Validate configuration before restart.
- Avoid exposing raw secrets in conversational output wherever possible.
